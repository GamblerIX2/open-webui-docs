## DDGS（Dux Distributed Global Search）

DDGS 是一个元搜索引擎，允许你通过统一接口搜索多个 provider。

### 设置

1.  在管理员设置中启用 **Web Search**
2.  将 **Web Search Engine** 设为 **DDGS**
3.  从下拉菜单中选择你想使用的 **DDGS Backend**

#### DDGS Backend 选项
- **Auto (Random)**：从可用 provider 中随机选择
- **Bing**
- **Brave**
- **DuckDuckGo**
- **Google**
- **Grokipedia**
- **Mojeek**
- **Wikipedia**
- **Yahoo**
- **Yandex**

#### 环境变量

你也可以通过 `DDGS_BACKEND` 环境变量配置 backend。

```bash
DDGS_BACKEND="google"
```
